Project 1

LuxeLoan AI — Credit Risk Prediction Engine

Next-gen credit risk predictor harnessing production-level Random Forest ML pipelines to determine loan approvals, evaluate structural asset ratios, and assess CIBIL credit ratings in real-time.

Deployed Cloud Hosted
Problem statement

Automate loan approval decisions by predicting credit risk from applicant financial profiles, asset valuations, and CIBIL scores — replacing slow manual underwriting with instant ML-driven assessments.

Dataset / data source

Structured loan application dataset with features including income, dependents, education, employment status, asset valuations (residential, commercial, luxury, bank), and CIBIL credit scores.

Model / approach

Random Forest classifier with stratified k-fold cross-validation, feature engineering on asset ratios, loan-to-income metrics, and CIBIL score classification bands.

Tech stack

Python, Scikit-learn, Flask, HTML/CSS/JS, Vercel

Cloud deployment

Vercel (serverless deployment with live public URL)

Results

98.36% prediction accuracy with real-time risk assessment dashboard and multi-metric output

Key features
  • Real-time loan approval prediction with circular progress visualization and confidence scoring.
  • Multi-asset portfolio analysis evaluating residential, commercial, luxury, and liquid bank assets side-by-side.
  • Interactive CIBIL credit score slider with live classification (Poor → Excellent) and risk assessment.
  • Dashboard-style results with loan-to-income ratio, asset coverage multiplier, and actionable recommendations.

Project 2

AI Delivery Time Predictor

ML-powered delivery time estimation engine that analyzes distance, traffic density, weather conditions, vehicle type, and courier experience to generate highly accurate real-time delivery predictions.

ML App Production Ready
Problem statement

Predict food delivery times accurately by modeling real-world variables — distance, weather, traffic levels, time of day, vehicle type, prep time, and courier experience — to optimize logistics and improve customer satisfaction.

Dataset / data source

Historical delivery records with multi-dimensional features including distance (km), weather conditions, traffic density, time-of-day segments, vehicle categories, preparation time, and courier experience levels.

Model / approach

Scikit-learn regression pipeline with feature encoding for categorical variables (weather, traffic, vehicle, time of day) and numerical inputs (distance, prep time, experience) for delivery time estimation.

Tech stack

Python, Scikit-learn, Flask, HTML/CSS/JS, Vercel

Cloud deployment

Vercel (serverless deployment with live public URL)

Results

Real-time delivery time predictions with model confidence scoring and historical pattern analysis

Key features
  • Multi-factor prediction combining 7 input variables — distance, weather, traffic, time of day, vehicle, prep time, and courier experience.
  • Glassmorphism UI with animated blob backgrounds and a polished dashboard-style result display.
  • Model confidence meter with animated progress bar showing prediction reliability percentage.
  • Instant serverless inference returning estimated delivery time in minutes with clear visual feedback.
